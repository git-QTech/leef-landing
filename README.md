# Leef Browser Landing Page

The official, "No Slop" landing page for Leef Browser. Built with vanilla HTML, CSS, and JS.

## Features
- **Apple-style Scroll Reveals:** Smooth animations as you navigate.
- **Privacy First:** Detailed breakdown of Leef's privacy-focused architecture.
- **Technical Transparency:** Clear explanation of the Electron + Chromium foundation.
- **Developer Focus:** Dedicated section for the core QTech team.

## Tech Stack
- **HTML5:** Semantic structure for SEO.
- **CSS3:** Custom design system with "Deep Forest" color palette.
- **JavaScript:** Intersection Observer API for performance-first animations.

## Deployment
This page is hosted via GitHub Pages.
