Release Information

Leef Browser follows a rapid release cycle. Stable builds are thoroughly tested for daily use, but you can also get it from GitHub and check our source code. We want to be completely open about how everything works.

System Requirements

  • Windows 10 or later (64-bit)
  • macOS 11.0 or later (Coming Soon)
  • Minimum 4GB RAM recommended
  • Note: Native ARM support is not planned; Apple Silicon is supported via Rosetta 2.

Installation & Build

Download & Install
Windows
  1. Click the Download for Windows button on this page.
  2. Run the .exe installer once the download completes.
Windows SmartScreen: You may see a "Windows protected your PC" warning on first run. Click More Info → Run Anyway to proceed.
macOS
  1. Click the Download for macOS button on this page.
  2. Double-click the downloaded .dmg file to mount it.
  3. Drag the Leef icon into your Applications folder.
macOS Gatekeeper: Since we do not sign our builds with a paid Apple Developer certificate yet, you will need to right-click the app in your Applications folder and select Open on the first run to bypass the system check.
Build from Source
  1. Clone the repo: git clone https://github.com/git-QTech/leef.git
  2. Install dependencies: npm install
  3. Run in dev mode: npm start
  4. Build installer: npm run dist
